#Tiki Topple Game 🎮

## Overview
This is a digital implementation of the Tiki Topple board game created during a hackathon.

## Features
- 2–4 player support
- Action cards system
- Turn-based gameplay
- Scoring based on secret tiki positions

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open index.html in browser
